# Comunicación Backend — Carpeta de solicitudes

Esta carpeta contiene solicitudes formateadas para el equipo de backend (Supabase).
Cada archivo es una solicitud independiente que la IA del compañero puede leer y ejecutar.

## Cómo usar esta carpeta

Cuando el frontend necesite algo del backend, Claude genera un archivo con nombre:
```
backend-YYYY-MM-DD-nombre-descriptivo.md
```

El archivo contiene:
- **Contexto**: qué estamos construyendo y por qué se necesita esto
- **Qué se necesita**: la solicitud concreta (SQL, endpoint, policy RLS, etc.)
- **Código listo para ejecutar**: el SQL o configuración lista para copiar y pegar
- **Impacto esperado**: qué cambia en el front cuando esto esté listo
